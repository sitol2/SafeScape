BFP Chatbot Test Page

This page demonstrates the functionality of the BFP rule-based chatbot with quick questions and navigation features.

How to Test the Chatbot

  1. Click the chatbot button in the bottom right corner
  2. Try clicking one of the quick questions
  3. Try typing a question in the input field
  4. Press Enter or click the send button
  5. Try navigation questions like "Take me to the Kids Zone"

Features Implemented

  • Rule-based chatbot with predefined responses
  • Quick question buttons organized by category
  • Navigation functionality for website sections
  • API-driven quick questions from Django backend
  • Responsive design for all device sizes
